The fastest way to waste an evening on the **Kakobuy Spreadsheet** is to open [KakobuyPlus](https://www.kakobuyplus.com/en/products), scroll randomly, and add everything that looks good. I have done that. It leads to a cart full of similar hoodies, two pairs of shoes you did not plan for, and a shipping bill that makes the whole haul feel expensive.

The better method is simple: **search with intent, shortlist aggressively, verify before ordering**. This guide is the exact workflow I would use on KakobuyPlus in 2026.

## Rule #1: Pick One Goal Before You Open the Spreadsheet

Do not open the spreadsheet asking, "What should I buy?" Open it with one of these goals:

- Find one shoe model
- Build a 4-item first haul
- Browse budget basics under a set price
- Replace one wardrobe staple (hoodie, tee, bag, jacket)

One goal keeps you from turning a focused search into a shopping spiral.

## Rule #2: Use KakobuyPlus Filters Before You Scroll

KakobuyPlus is not just a giant list. It has real discovery tools:

- Keyword search for products or brands
- Category filter
- Brand filter
- Price range filter
- Color / gender / style filters

### Best search order

1. Choose a **category** first
2. Add a **brand** or keyword second
3. Narrow with **price range** third
4. Open only the rows that survived all three filters

### Example workflows

| Your goal | Start here |
| --- | --- |
| Black AF1 | Shoes → search `Air Force 1 black` |
| Budget tee | Clothing → price filter low → search `tee` |
| Crossbody bag | Bags → search `crossbody black` |
| Winter jacket | Clothing → search `puffer black` or `Arc'teryx` |
| Daily accessory | Accessories → filter by low price |

This is much more efficient than scrolling 110,000+ products from the top.

## Rule #3: Search Specific Keywords, Not Broad Brands

Broad searches create broad mistakes.

| Too broad | Better keyword |
| --- | --- |
| Nike | Dunk Low Panda |
| Jordan | AJ4 Military Black |
| Hoodie | Essentials fleece grey |
| Bag | Louis Keepall 50 |
| Jacket | Alpha SV black |
| Socks | Nike crew white 3 pack |

The more specific your keyword, the fewer false positives you have to inspect.

## Rule #4: Start with Hot / Recommended Items When You Are Unsure

If you do not know what to buy yet, begin with items marked **Hot** or shown in recommended sections on KakobuyPlus.

Why this helps:

- Popular listings usually have more buyer activity
- Sellers are more likely to still be active
- QC references are easier to find
- You learn what the market is actually buying right now

Hot items are not automatically perfect, but they are a better starting point than random deep-scroll finds.

## Rule #5: Build a Shortlist in Three Passes

### Pass 1: Collect 8–12 candidates

While searching, save anything that looks promising. Do not order yet.

### Pass 2: Cut to 4–6 items

Remove anything that:

- Has no size info
- Looks too cheap for the category
- Duplicates something you already shortlisted
- Comes from a platform you do not know how to order yet

### Pass 3: Final 3–5 items

Ask one question for each row:

> Would I still want this if shipping adds $8–$15 to the final cost?

If the answer is no, delete it.

This one question eliminates a surprising amount of impulse buying.

## Rule #6: Verify Every Link Before Kakobuy

A spreadsheet row is only useful if the live listing still works.

Check these before copying the URL:

| Check | Pass / fail |
| --- | --- |
| Product page opens | Must pass |
| Size or variant exists | Must pass |
| Price close to spreadsheet estimate | Should pass |
| Seller page looks active | Should pass |
| Platform identified (Taobao / Weidian / 1688) | Must pass |

If the page is dead, do not waste time trying to force it through Kakobuy. Go back to the spreadsheet and pick the next candidate.

## Rule #7: Know the Platform Before You Order

This saves beginners the most frustration.

| Platform | Typical Kakobuy action |
| --- | --- |
| Taobao | Paste link into search |
| 1688 | Paste link into search, watch MOQ |
| Tmall / JD | Paste link into search |
| Weidian | Often needs **Looking for Goods** manual form |
| Yupoo / Xianyu / WeChat seller | Manual order form |

If you found the item on KakobuyPlus but Kakobuy does not auto-load it, check the platform first before assuming the find is bad.

## Rule #8: Read the Small Notes Like They Matter

The best spreadsheet entries often include small details that are easy to ignore:

- `size up once`
- `budget batch`
- `thick fleece`
- `check logo`
- `fits oversized`
- `2026 updated link`

Those notes matter more than the hero photo. A jacket that looks oversized on the model may run short in the sleeves. A shoe listed as budget may be fine for daily wear but weak on details.

## Search Methods Compared

| Method | Best for | Weakness |
| --- | --- | --- |
| Category + filter | First haul, budget browsing | Less precise for one exact model |
| Keyword search | Exact item hunt | Can still return too many matches |
| Hot / recommended | Beginners, trend discovery | Not ideal for niche sizes or rare colorways |
| Brand filter | Loyalty to one label | Can hide better value from other sellers |
| Old spreadsheet Ctrl+F | Legacy Google Sheet files | Slower and less structured than KakobuyPlus |

If you are on KakobuyPlus, site search and filters should be your default. Keep `Ctrl + F` as a backup only if you are still using an old shared sheet file.

## Red Flags to Skip Immediately

- Dead or redirected product page
- No sizes available
- Price far below normal for that item type
- Only seller photos, no QC or buyer references
- Same item added to your list in three similar versions
- Complex shoe with zero QC history
- You cannot tell whether it is Taobao or Weidian

## FAQ

### How many items should I shortlist before ordering?

I would shortlist 8–12, then order 3–5. That is enough choice without creating a messy first haul.

### Is it better to search by brand or by item type?

Item type first, brand second. "Black puffer jacket" is usually better than starting with only "Moncler."

### What is the biggest search mistake beginners make?

Scrolling the full catalog with no category, no price limit, and no shortlist process.

## Bottom Line

Finding items on the **Kakobuy Spreadsheet** is not about luck. It is about using KakobuyPlus the right way: one goal, category filters, specific keywords, Hot items as a starting point, aggressive shortlist cuts, and live-link verification before Kakobuy.

Search like you are building a cart on purpose — not like you are window shopping with a credit card nearby.
