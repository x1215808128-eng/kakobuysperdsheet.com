QC photos are only useful if you act on them. Many **Kakobuy Spreadsheet** buyers approve too fast — then discover wrong sizes or flawed stitching after paying $50+ international freight. Returns and exchanges are **cheap inside China** and **expensive after export**.

This guide explains when to reject, exchange, or request refunds on Kakobuy, how seller return windows work, and how to keep spreadsheet hauls on track in 2026.

## Returns vs. Exchanges: Know the Difference

| Action | When to use | Typical outcome |
| --- | --- | --- |
| **Exchange** | Wrong size, wrong color variant, minor fixable mismatch | Seller sends replacement to warehouse |
| **Return / reject** | Major defect, wrong model, unacceptable quality | Refund to Kakobuy balance (minus fees) |
| **Retake QC photos** | Dark angles, missing shots — item may be fine | New photos before you decide |

**Golden rule:** if QC fails, resolve it **before** consolidation and shipping authorization.

## Why Spreadsheet Buyers Get Stuck

Spreadsheet rows on [KakobuyPlus](https://www.kakobuyplus.com/en/products) show curated finds — not guaranteed batch perfection. Common failure points:

- Seller sent a cheaper substitute model
- Size label does not match your order (EU vs US sizing)
- Batch stitching differs from spreadsheet hype photos
- Seller shipped wrong colorway

QC is your last cheap checkpoint. Once the parcel crosses borders, you usually absorb return shipping or sell locally at a loss.

## Step-by-Step: Reject or Exchange on Kakobuy

### 1. Review all QC photos first

Use the full [QC Photos Guide](/kakobuy-spreadsheet-news/kakobuy-qc-photos-guide-2026) workflow — scroll every angle before clicking anything.

### 2. Open the order line in Kakobuy

Navigate to **Warehouse** → select the item → choose:

- **Exchange** — for size/color swaps when seller allows
- **Return** — for defects or wrong item
- **Request more photos** — if you cannot judge quality

### 3. Add a clear note

Write specific issues:

- "Ordered US 9, tag shows EU 42 / wrong size"
- "Logo stitching misaligned on left shoe"
- "Received black, ordered navy per listing"

Vague notes slow agent communication with Chinese sellers.

### 4. Wait for seller response

Timelines vary by seller and platform (Taobao, Weidian, 1688):

- Fast sellers: 1–3 business days
- Slow or holiday periods: up to 7+ days
- Some listings have **no return policy** — read before you order

### 5. Track replacement QC

Exchanged items get a **new QC cycle**. Do not assume the replacement is perfect — review photos again.

## Return Windows and Fees

Community experience with major agents in 2026:

| Scenario | Typical handling |
| --- | --- |
| Domestic return to seller | Low China postage — agent coordinates |
| Seller refuses return | Partial refund or dispute — case by case |
| Kakobuy service fee | May apply on some returns — check order page |
| International return after export | Rarely economical — avoid if possible |

**7-day buyer protection** style windows often apply to certain marketplace orders — confirm in Kakobuy's order UI for your specific line item. Do not wait until day six if photos already show a wrong model.

## When to Exchange vs. Reject

| Situation | Best action |
| --- | --- |
| Wrong size, same model available | Exchange |
| Slight color difference, wearable | Personal choice — exchange or approve |
| Wrong batch / wrong shoe model | Reject |
| Damaged sole, torn fabric | Reject |
| Missing accessories (laces, dust bag) | Message agent — exchange or partial refund |
| Spreadsheet link ≠ received item | Reject + note listing mismatch |

## Protecting Multi-Item Spreadsheet Hauls

If one item in a consolidated cart fails QC:

1. **Do not ship the whole parcel** until the problem line is resolved or removed
2. **Remove rejected items** from consolidation draft
3. **Recalculate** shipping weight after exchanges (size changes can alter box plan)
4. **Update customs declaration** if final cart changes — see [Customs Guide](/kakobuy-spreadsheet-news/kakobuy-customs-declaration-guide-2026)

Shipping half-approved / half-disputed carts creates declaration mismatches and wasted freight.

## Insurance and Agent Support

For higher-value spreadsheet finds:

- Consider **shipping insurance** at export — separate from return policy
- Forward seizure or damage notices to Kakobuy support quickly
- Keep screenshots of QC flaws and seller listing pages

Reputable agents document disputes with sellers on your behalf — but they need clear evidence from day one.

## Common Return Mistakes

- **Approving then trying to return** after international ship
- **Ignoring seller "no return" tags** on spreadsheet links
- **Exchanging without checking replacement QC**
- **Rushing during sale events** when sellers are slow to respond
- **Consolidating before all QC lines are green**

## FAQ

**Can I return after shipping internationally?**  
Practically no for most buyers — cost exceeds item value.

**Does Kakobuy refund to PayPal automatically?**  
Refunds typically return to your Kakobuy balance first — withdrawal rules depend on account settings.

**What if the seller ignores the return?**  
Escalate through Kakobuy support with QC evidence and order timestamps.

**Should I reorder from a new spreadsheet link instead?**  
Sometimes faster than fighting a stubborn seller — compare return deadline vs. replacement ETA.

## Bottom Line

The **Kakobuy Spreadsheet** accelerates discovery. **Returns and exchanges** protect you between warehouse and doorstep. Treat QC as a decision gate — reject bad lines early, exchange sizes promptly, and never authorize export until every item in your cart matches what you intended to buy.

That habit saves more than any spreadsheet discount code.
